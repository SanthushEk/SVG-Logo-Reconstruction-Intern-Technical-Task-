SVG Custom Shape Project
📌 Overview

This project demonstrates how to create a custom shape using SVG (Scalable Vector Graphics).
It was developed as an Intern Technical Task for Perpova Developers.

The project focuses on understanding:

SVG structure

Coordinate systems

Path drawing commands

Vector shape creation

📄 Project Description

The project contains:

A 600 × 600 drawing canvas

White background

Custom red curved shape created using SVG path commands

🧠 Technologies Used

SVG (Scalable Vector Graphics)

XML Syntax

HTML Compatible Vector Graphics
